Дополнительные библиотеки:

В программе дополнительно используются библиотеки:
-SOIL для загрузки и использования текстур
    Установка: sudo apt-get install libsoil-dev
-GLM для упрощения работы с матрицами и векторами
    Установка: sudo apt-get install libglm-dev
-ASSIMP для работы с mesh файлами
    Установка: sudo apt-get install libassimp-dev assimp-utils
