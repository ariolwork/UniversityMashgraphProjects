Управление:

esc    -    закрытие
0/1    -    сцена с фигурами
2      -    сцены с фрактальными поверхностями(при повторном нажатии кнопки 2 
            поверхности меняются. Всего их 4)
3      -    включение круговой анимации камеры(камера начинает вращаться 
            вокруг сцены)
4      -    регулировка уровня тумана(зацикленна)
5      -    регулировка уровня теней(зацикленна) Соответственно мягкие и жесткие
            тени регулируются тут же
wasdrf -    движение по сцене
qe     -    вращение камеры
shift  -    ускорение движения
мышь   -    вращение камеры
